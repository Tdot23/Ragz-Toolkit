# Ragz-Toolkit
Ragz toolkit is a combination of various tools for testing my home SIEM systems
# Installation
git clone https://github.com/Tdot23/Ragz-Toolkit.git 


cd Ragz-Toolkit

chmod +x setup.sh


chmod +x RAGZ

sudo ./setup.sh

# Usage
sudo ./RAGZ
